## Características do PaperMod

O tema PaperMod oferece:

- Design limpo e moderno
- Suporte a modo escuro/claro
- Responsivo
- Rápido carregamento
- SEO otimizado

## Código de exemplo

```python
def hello_world():
    print("Olá, mundo!")
    return "Hugo + PaperMod é incrível!"

if __name__ == "__main__":
    message = hello_world()
    print(message)
```
